# Vikas Thakur - Frontend Developer Portfolio

A modern, minimal, and premium portfolio website showcasing frontend development skills and projects. Built with vanilla HTML, CSS, and JavaScript without any external libraries or frameworks.

## Features

### Design
- **Modern & Minimal Design** - Inspired by Apple, Linear, and Vercel aesthetics
- **Dark/Light Mode** - Toggle between themes with system preference detection
- **Fully Responsive** - Optimized for mobile, tablet, laptop, and desktop
- **Accessibility Friendly** - WCAG compliant with keyboard navigation support
- **Premium Typography** - Inter font family with optimized font weights

### Sections
1. **Hero** - Professional introduction with typing animation
2. **About** - Personal bio with animated statistics counter
3. **Skills** - Technology stack with progress indicators
4. **Projects** - Project showcase with hover effects
5. **Learning Journey** - Timeline of career milestones
6. **Education** - Academic background
7. **Certifications** - Professional credentials
8. **Contact** - Contact form with validation
9. **Footer** - Links and copyright

### Interactions
- Smooth scroll reveal animations
- Typing text animation effect
- Animated counters with easing
- Sticky navigation with active state
- Scroll-to-top button
- Mobile navigation menu
- Form validation with feedback
- Hover effects and micro-interactions

## Tech Stack

- **HTML5** - Semantic markup with accessibility attributes
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** - ES6+ features, no dependencies

## Getting Started

### Local Development

1. Clone or download the repository
2. Open `index.html` in a browser
3. No build process required

### Deployment

Upload all files to your hosting provider or use platforms like:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Customization

### Personal Information
Edit `index.html` to update:
- Name and title
- Social media links
- Project details
- Contact information

### Styling
Modify `css/style.css` to change:
- CSS custom properties for colors
- Typography settings
- Spacing and layout values

### Content
Update sections in `index.html`:
- Projects grid
- Timeline items
- Skills and certifications
